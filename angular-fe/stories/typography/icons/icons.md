# Icons

## Usage

```html
<icon glyph="bookmark" size="default" bg="false"></icon>
```

## Properties

| Name  | Default  | Values  |  Type | Description  |
|---|---|---|---|---|
| glyph | - | ... | string | Icon name
| size | default | default, large, medium | string | Size
| bg | false | true, false | boolean | Background
