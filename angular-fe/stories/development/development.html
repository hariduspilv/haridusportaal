Installation

Clone project from: https://github.com/hariduspilv/haridusportaal

Run `npm install` to install projects dependencies

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

Run `npm run storybook` to start developing in storybook

Run `ng build --prod` to build app dist files

Run `npm run storybook` to build storybook dist files
