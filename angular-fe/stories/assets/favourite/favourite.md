# Favourite

## Usage

```html
<favourite 
  id="48788"
  state="true"
  title="Artikkel mugudest">
</favourite>
```

## Properties

| Name  | Default  | Values  |  Type | Description  |
|---|---|---|---|---|
| id | - | - | number | Content Nid (unique identifier)
| state | - | true, false | boolean | State
| title | - | - | string | Content title
