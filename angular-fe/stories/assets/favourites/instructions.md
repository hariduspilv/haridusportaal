# Favourites

## Usage

```html
<favourite 
  id="48788"
  state="true">
</favourite>
```

## Properties

| Name  | Default | Values      | Type    | Description                                                |
|-------|---------|-------------|---------|------------------------------------------------------------|
| id    | -       | -           | number  | Content Nid (unique identifier)                            |
| state | -       | true, false | boolean | State                                                      |
| limit | -       | true, false | boolean | Boolean indicating if favourites limit reached (storybook) |
