# Charts

## Usage

```html
<chart
  [data]='data'
  [type]='type'>
</chart>
```

## Properties

| Name  | Default  | Values  |  Type | Description  |
|---|---|---|---|---|
| data | - | - | Object[] | Chart data
| type | - | 'filter', - | string | Filters or no
